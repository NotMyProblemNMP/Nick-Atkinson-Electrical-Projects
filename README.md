# Nick Atkinson Electrical Projects — Website

Premium dark-themed marketing landing page for Nick Atkinson Electrical
Projects Limited, a NAPIT-approved electrician based in North Shields covering
Newcastle upon Tyne and the North East.

Built with **Next.js (App Router) + Tailwind CSS + Framer Motion**. Fully
static output — ideal for Vercel's free tier.

## Enquiry form

The final-CTA form works two ways:

- **No setup (default):** submitting opens the visitor's email app with the
  enquiry pre-filled, addressed to `site.email`.
- **Recommended:** create a free access key at [web3forms.com](https://web3forms.com)
  pointed at Nick's inbox, then set the environment variable
  `NEXT_PUBLIC_WEB3FORMS_KEY` in Vercel → Project → Settings → Environment
  Variables. The form then sends directly from the page.

## Local development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
```

## Deploying to GitHub + Vercel

1. Create a GitHub repository and push:

   ```bash
   git remote add origin https://github.com/<your-account>/nick-atkinson-electrical.git
   git push -u origin main
   ```

2. Go to [vercel.com/new](https://vercel.com/new), import the repository and
   click **Deploy** — no configuration needed (Next.js is auto-detected).

3. When the domain is registered, add it under Vercel → Project → Settings →
   Domains, and update `url` in `lib/site.ts`.

## SEO / AI-search notes

- Full metadata + Open Graph + Twitter cards in `app/layout.tsx`
- `Electrician` JSON-LD structured data (rating, reviews, services, areas
  served) in `lib/jsonld.ts` — this is what makes the business quotable by
  Google rich results and AI answer engines
- `sitemap.xml` and `robots.txt` generated by `app/sitemap.ts` / `app/robots.ts`
- Once live, submit the sitemap in
  [Google Search Console](https://search.google.com/search-console)
